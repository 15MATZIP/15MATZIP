# 항해 15조 웹 미니 프로젝트

## MAT.ZIP

> 사용자들의 맛집정보를 공유하는 간단한 게시판입니다.
> CRUD 를 경험할 수 있었던 프로젝트 였고, 팀원과의 협업을 경험해 볼 수 있었던 프로젝트였습니다.

---

## 실행 설명

- 실행 파일 이름 : app.py
  - Mac

```
python3 app.py
```

- Window

```
python app.py
```

---

## 기능

1. Create

- 회원가입 기능
- 게시글을 생성하는 기능 (제목, 주소, 별점, 이미지 URL, 게시글)

2. Read

- 게시글들을 가져오는 기능

3. Update

- 게시글을 수정하는 기능 (제목, 주소, 별점, 이미지 URL, 게시글 변경)

4. Delete

- 유저가 작성한 게시글을 삭제하는 기능
